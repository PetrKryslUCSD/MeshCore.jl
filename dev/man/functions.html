Topology operations

MeshCore.skeleton — Function
skeleton(shapes::ShapeCollection; options...)

Compute the skeleton of the shape collection.

It consists of facets (shapes of manifold dimension one less than the manifold dimension of the shapes themselves).

Options

  • boundaryonly: include in the skeleton only shapes on the boundary of the input collection, true or false (default).
source
MeshCore.boundary — Function
boundary(shapes::ShapeCollection)

Compute the shape collection for the boundary of the collection on input.
